# CoreXY Structure 3D Printer
## Overview
- Core XY structure design, full belt drive, higher stability and printing accuracy, ensure the quality of the model; 
- Dual Z-axis design, increase the bearing capacity of the hot bed, reduce errors in printing, and print more finely. 
- Customizable print range, automatic creation of drawings and parts with Siemens NX
- Capable of building the ideal 3D printer for a beginner. Additionally, it is remarkably convenient for future part upgrades.
